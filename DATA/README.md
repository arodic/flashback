# Game Data Files

This folder should contain the original Flashback PC DOS game files.

## Required Files

| Extension | Count | Description |
|-----------|-------|-------------|
| `*.CMD` | 29 | Cutscene command bytecode |
| `*.POL` | 29 | Cutscene polygon/vertex data |
| `*.MID` | 11 | MIDI music files |
| `*.PRF` | 28 | Instrument configuration |
| `*.INS` | 29 | AdLib instrument patches |

## How to Obtain

Purchase Flashback from:
- [GOG.com](https://www.gog.com/game/flashback)
- [Steam](https://store.steampowered.com/app/961620/Flashback/)

Then copy the contents of the game's `DATA/` folder here.

## File Locations

- **GOG**: `<install path>/DATA/`
- **Steam**: `<Steam>/steamapps/common/Flashback/DATA/`
